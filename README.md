# 🌐 FABLIBRAS — Landing Page

Landing page oficial da plataforma **FABLIBRAS**, desenvolvida com foco em acessibilidade digital e inclusão através da Língua Brasileira de Sinais (Libras).

O projeto apresenta uma interface moderna, responsiva e intuitiva, reforçando a importância da acessibilidade na web e da democratização da comunicação para pessoas surdas.

---

## 🚀 Deploy do Projeto

### Acesse a aplicação online: 👇

### [Landing Page FabLibras](https://page.fablibras.com)

## 📖 Sobre o Projeto

A FABLIBRAS foi criada para demonstrar uma proposta de plataforma acessível, utilizando uma experiência visual moderna e organizada para apresentar funcionalidades, objetivos e impacto social do projeto.

A landing page foi construída pensando em:

- Inclusão digital
- Acessibilidade web
- Responsividade
- Boa experiência do usuário (UI/UX)
- Performance e design moderno
